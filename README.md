# Student-record-management-system
A Student Record Management System developed in C for storing and managing student records.
## Description

A menu-driven file-based Student Record Management System developed in C for storing and managing student records using structures and file handling.

## Features

* Add Student Records
* Display Student Records
  - Search by Name
  - Search by Roll Number
  - Search by Department
  - Search by Year
* Update Student Records
* Delete Student Records

## Concepts Used

* Structures
* File Handling
* fread()
* fwrite()
* fseek()
* Function Pointers
